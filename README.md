## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`


Problem Statement
Traditional library management systems primarily rely on keyword-based searches and manual processes for book management, issue/return tracking, fine calculation, and recommendation generation. As library collections grow, users often struggle to discover relevant books, while librarians face challenges in managing inventory, monitoring overdue books, and analyzing borrowing patterns. Existing systems lack intelligent recommendation capabilities and fail to capture meaningful relationships between books, authors, genres, and users, resulting in poor user experience and inefficient resource utilization.


Proposed Solution
The AI-Powered Library Management System with Knowledge Graph addresses these challenges by combining Machine Learning, Natural Language Processing (NLP), and Knowledge Graph technology to create an intelligent and automated library ecosystem.

Key Solutions
📚 Intelligent Library Management
Automates book cataloging, issuing, returning, and inventory tracking.
Reduces manual effort and administrative workload.
🔍 Semantic Book Search
Uses Sentence Transformers to understand the meaning of user queries.
Enables users to search books using natural language instead of exact keywords.
🤖 Personalized Recommendation System
Recommends books based on borrowing history, genre preferences, and similar user behavior.
Improves book discovery and user engagement.
🕸 Knowledge Graph-Based Relationships
Builds connections among books, authors, categories, publishers, and users.
Helps identify similar books and hidden relationships for better recommendations.
💰 Automated Fine Management
Automatically calculates overdue fines.
Tracks payment status and generates fine reports.
📊 Analytics and Insights
Provides dashboards for monitoring library usage, popular books, overdue records, and user activity.
Supports data-driven decision-making for librarians and administrators.
🔒 Secure Role-Based Access
Separate functionalities for Admins, Librarians, and Students.
Ensures secure and controlled access to library resources.
Expected Impact
Reduce manual library operations by 90%+.
Improve book discovery through AI-powered semantic search.
Increase recommendation relevance and user satisfaction.
Enhance inventory utilization and resource management.
Provide a modern, scalable, and intelligent library ecosystem.
